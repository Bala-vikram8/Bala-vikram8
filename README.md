## 📌 Featured Projects

### 🔹 [Fin Agent Shield](https://github.com/Bala-vikram8/Finnagent-Shield)
Production-grade reliability and security framework for AI agents in regulated financial environments. Built with **Python, FastAPI, and Claude API** using agentic workflow design patterns. Implements a three-layer protection system covering operational failures, reasoning failures, and adversarial attacks (prompt injection, data validation) — with failure classification, circuit breaking, and compliance audit logging.

### 🔹 [Predictive Fleet Maintenance & Risk Dashboard](https://github.com/Bala-vikram8/Fleet-Risk-Prediction-Analytics)
Analyzed **350K+ vehicle records** and built ML models (**Random Forest, XGBoost**) in R to predict high-risk assets and repair costs. Delivered an interactive **Tableau** dashboard enabling drill-down by location, asset type, and year — empowering smarter budgeting and early risk detection for fleet managers.

### 🔹 [NYC Traffic Crash Analysis — TDSP Safety Project](https://github.com/Bala-vikram8/TDSP_Crash_Analysis)
Analyzed **1M+ crash records** using **Python, R, and Power BI** as part of the Transportation Data Science Project. Identified peak risk zones and top crash causes (40% from inattention) and borough-level injury patterns, presented via a scientific poster in collaboration with the **USDOT** and National Student Data Corps.
